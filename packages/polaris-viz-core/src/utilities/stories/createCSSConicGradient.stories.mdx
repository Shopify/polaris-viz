<Title type="h3">
  <code>createCSSConicGradient()</code>
</Title>

<p>Returns a css conic gradient from the provided gradient.</p>

```tsx
const gradient: gradientStop[] = [
  {color: 'red', offset: 0},
  {color: 'green', offset: 100},
];

const conicGradient = createCSSConicGradient(gradient);

// conic-gradient(red 0%, green 100%)
```
