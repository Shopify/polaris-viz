<Title type="h3">
  <code>changeColorOpacity()</code>
</Title>

<p>
  Changes the color to have the provided opacity. Accepts any color format,
  returns an `rgba` string.
</p>

```tsx
changeColorOpacity('#ffffff', 0.5);
// 'rgba(255, 255, 255, 0.5)'
```
