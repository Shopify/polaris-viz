<Title type="h3">
  <code>clamp()</code>
</Title>

<p>Is used to keep a number within the inclusive range within the min and max bounds. This method returns the clamped amount.</p>

```tsx
clamp({
  amount: 10,
  min: 15,
  max: 20,
});

// Output:
// 15


clamp({
  amount: 1000,
  min: 150,
  max: 200,
});

// Output:
// 200
```
