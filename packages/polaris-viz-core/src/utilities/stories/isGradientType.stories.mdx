<Title type="h3">
  <code>isGradientType()</code>
</Title>

<p>
  Returns a <code>boolean</code> based on if the provided color is a gradient or
  solid color.
</p>

```tsx
isGradientType([
  {offset: 0, color: 'red'},
  {offset: 100, color: 'green'},
]);

// true

isGradientType('red');

// false
```
