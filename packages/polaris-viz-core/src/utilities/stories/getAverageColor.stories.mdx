<Title type="h3">
  <code>getAverageColor()</code>
</Title>

Given 2 color strings, it returns the average of them; the resulting color of mixing both.

```tsx
// mixing yellow and blue:

getAverageColor('rgb(255, 227, 0)', 'rgb(0, 137, 255)');

// returns green:
// 'rgb(128,182,128)'
```
