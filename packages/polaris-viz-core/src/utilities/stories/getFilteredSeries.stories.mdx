<Title type="h3">
  <code>getFilteredSeries()</code>
</Title>

<p>Returns an array where all comparison data has been removed.</p>

```tsx
const DATA = [
  {
    data: [{key: 0, value: 100}],
  },
  {
    isComparison: true,
    data: [{key: 0, value: 200}],
  },
];

const series = getFilteredSeries(DATA);

// [
//   {
//     data: [{key: 0, value: 100}],
//   },
// ]
```
