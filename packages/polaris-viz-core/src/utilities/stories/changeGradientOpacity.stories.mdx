<Title type="h3">
  <code>changeGradientOpacity()</code>
</Title>

Similar to `changeColorOpacity`, but changes all colors of `GradientStop[]` to
the provided target opacity

```tsx
changeGradientOpacity(
  [
    {color: 'rgb(255, 0, 0)', offset: 1},
    {color: 'rgb(123, 123, 123)', offset: 2},
  ],
  0.5,
);

// [
//   {color: 'rgba(255, 0, 0, 0.5)', offset: 1},
//   {color: 'rgba(123, 123, 123, 0.5)', offset: 2},
// ]
```
