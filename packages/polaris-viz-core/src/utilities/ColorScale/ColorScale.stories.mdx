<Title type="h3">
  <code>ColorScale()</code>
</Title>

<img src="color-scale.png" alt="Grid displaying all available color scales" />

<p>
  Returns a scale function that returns a backgroundColor and a textColor,
  according to the Polaris Viz Color Matrix:
</p>

```tsx
const tealScale = ColorScale(Hue.Teal, 0, 100);

tealScale(0);

// {
//   backgroundColor: '#f0f8fb', // colorTeal00 from polaris-viz-core
//   textColor: 'colorGray150'
// }

tealScale(100);

// {
//   backgroundColor: '#001419', // colorTeal160 from polaris-viz-core
//   textColor: 'white'
// }
```
