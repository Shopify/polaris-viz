  In the <code>Theme</code> type definition, you'll find multiple properties;

  specific chart type.
</p>

```tsx
export interface Theme {
  chartContainer: ChartContainerTheme;
  bar: BarTheme;
  grid: GridTheme;
  xAxis: XAxisTheme;
  yAxis: YAxisTheme;
  line: LineTheme;
  crossHair: CrossHairTheme;
  legend: Legend;
  seriesColors: SeriesColors;
  tooltip: TooltipTheme;
}
```

Navigate the `Chart Specific Properties` and `Global Properties` sections to see examples of
each property in action.
