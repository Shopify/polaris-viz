<Title type="h2">Installation</Title>

Install the package by running:

```bash
# If you use npm:
npm install @shopify/polaris-viz-native --save

# If you prefer Yarn, use the following command instead:
yarn add @shopify/polaris-viz-native
```

Wrap the top-level component of your app with `PolarisVizProvider`

<Banner type="Alert">
  Remember to pass <code>native</code> to the provider otherwise the app won't
  work!
</Banner>

```js
import {PolarisVizProvider} from '@shopify/polaris-viz';

export const YourTopLevelComponent = () => {
  return (
    <PolarisVizProvider native>
      <App />
    </PolarisVizProvider>
  );
};
```

<Title type="h2">Peer dependencies</Title>

Polaris Viz has peer dependencies on:

- `react@^16.8.6`
- `react-native@^0.58`

You are responsible for providing these packages in your project. By requiring these packages as `peerDependencies` we can be sure there won't be duplicate packages included due to version mismatches.
