<Title type="h2">Installation</Title>

Install the package by running:

```bash
# If you use npm:
npm install @shopify/polaris-viz --save

# If you prefer Yarn, use the following command instead:
yarn add @shopify/polaris-viz
```

<Title type="h3">Set up PolarisVizProvider</Title>

Wrap the top-level component of your app with the PolarisVizProvider.

<Banner type="Alert">
  The PolarisVizProvider is responsible for managing{' '}
  <a href="http://polaris-viz.shopify.io/iframe.html?id=shared-themes-default-themes--page&args=&viewMode=story">
    Themes
  </a>
  , as well as providing the correct elements and animation functions for web development.
  Your app won't work without it.
</Banner>

```js
import {PolarisVizProvider} from '@shopify/polaris-viz';

export const YourTopLevelComponent = () => {
  return (
    <PolarisVizProvider>
      <App />
    </PolarisVizProvider>
  );
};
```

<Title type="h2">CSS Styles</Title>

If your app is set up to consume esnext builds, import components where you want
to use them and the styles will automatically be applied.

If your app is not set up for esnext, import the styles where you import components from the library:

```js
import '@shopify/polaris-viz/build/esm/styles.css';
```

<Title type="h2">Peer dependencies</Title>

Polaris Viz has `react@^16.8.6` as a peer dependency.

You are responsible for providing this package in your project. By requiring this package as `peerDependencies` we can be sure there won't be duplicate packages included due to version mismatches.
