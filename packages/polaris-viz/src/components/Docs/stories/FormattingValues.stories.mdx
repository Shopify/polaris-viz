<Title>üè≠ Formatting Values</Title>

All charts that display values as text offer options to customize the way you can format them.

<Divider noLine />
<Title type="h2">Axis Values</Title>

You can pass in a custom function to format the values on the axis.

<ExamplesGrid cols={1}>
  <ComponentContainer
    codeSample={`
  <LineChart
  theme="Light"
  yAxisOptions={{
    labelFormatter: (value) => {
      return \`\${value} pickles\`;
    },
  }}
  xAxisOptions={{
    labelFormatter: (value) => {
      return \`\${new Date(value).toLocaleDateString('en-US', {
        dateStyle: 'medium',
      })}\`;
    },
  }}
  data={[
    {
      name: 'Sales',
      data: [
        {value: 13, key: '2022-01-02'},
        {value: 42, key: '2022-01-03'},
      ],
    },
  ]}
  showLegend={false}
/>
    `}
    theme="Light"
    center
    chart={
      <div
        style={{
          width: '100%',
          display: 'block',
          position: 'absolute',
          height: '140px',
          paddingTop: '20px',
        }}
      >
        <PolarisVizProvider
          themes={{
            Default: {
              chartContainer: {
                borderRadius: '0',
                padding: '0',
                backgroundColor: 'black',
              },
            },
          }}
        >
          <LineChart
            theme="Light"
            yAxisOptions={{
              labelFormatter: (value) => {
                return `${value} pickles`;
              },
            }}
            xAxisOptions={{
              labelFormatter: (value) => {
                return `${new Date(value).toLocaleDateString('en-US', {
                  dateStyle: 'medium',
                })}`;
              },
            }}
            data={[
              {
                name: 'Sales',
                data: [
                  {value: 13, key: '2020-01-02'},
                  {value: 42, key: '2020-01-03'},
                ],
              },
            ]}
            showLegend={false}
          />
        </PolarisVizProvider>
      </div>
    }
  />
</ExamplesGrid>
