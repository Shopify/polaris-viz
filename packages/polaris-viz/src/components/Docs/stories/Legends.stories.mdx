<Title>ðŸ““ Legends</Title>

<p>
  The more data series in a chart, the more colors and shapes it has. This makes
  it more challenging to parse the information quickly. By adding interactive
  legends we can highlight individual series to allow users to parse information
  in another way and helps distinguish between series for people with color
  vision deficiencies.
</p>

<SampleLegendChart />

<p>
  Legends respond to <strong>hover</strong> and <strong>focus</strong> events to
  highlight the corresponding series on the chart.
</p>

<Title type="h2">Guidelines</Title>

<ul>
  <li>Legends are not available on SparkCharts.</li>
  <li>Legends should always be used on charts with more than 1 series.</li>
</ul>

<Title type="h3">How are Legends Generated?</Title>

<p>
  Legend <code>title</code> and <code>color</code> are generated from the root
  <code>name</code> properties from the provided <a href="/docs/docs-data-structure--page">
    DataSeries
  </a>.
</p>

<SampleLegendContainer />

```tsx
const DATA = [
  {
    name: 'Breakfast',
    data: [...]
  },
  {
    name: 'Lunch',
    color: 'green',
    data: [...]
  },
  {
    name: 'Dinner',
    data: [...]
  },
];
```
