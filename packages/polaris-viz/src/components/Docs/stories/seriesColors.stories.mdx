<Title type="h3">
  <code>seriesColors</code>
</Title>

`seriesColors` allows you to customize what colors the charts will use depending on how many data series are being displayed.

The `Default` and `Light` themes that come with the library out of the box use gradients for charts with up to 7 data series, but use
solid colors for charts with more. The `Color` type accepts a gradient definition or any valid CSS string color:

```ts
export type Color = string | GradientStop[];

export interface GradientStop {
  offset: number;
  color: string;
}
```

<PropertyTable global>

  <PropertyTable.Row

      </>
    }
  />
  <PropertyTable.Row
    property={`seriesColors.all`}
    type={'Color[]'}
    description={

    }
  />
